# キー押しっぱなしツール

Enterキーや任意のキーを「押しっぱなし」または「連打」状態にできる小さなデスクトップアプリです。

## 機能

- **押しっぱなし(ホールド)**: 選んだキーを停止するまで押し続けます
- **連打(リピート)**: 指定した間隔でキーを連打します(間隔は秒で指定、例: 0.05)
- **F6キーでON/OFF切替**: 他のアプリを操作中でも F6 でどこからでも開始/停止できます
- 選べるキー: Enter / Space / Shift / Ctrl / Alt / Tab / 矢印キー / W・A・S・D

## 使い方

### 1. Pythonのインストール(未インストールの場合)

https://www.python.org/downloads/ から最新版をインストールしてください。
インストール時に **「Add Python to PATH」に必ずチェック** を入れてください。

### 2. ライブラリのインストール(初回のみ)

コマンドプロンプトで:

```
pip install pynput
```

### 3. 起動

`key_hold.py` をダブルクリック、またはコマンドプロンプトで:

```
python key_hold.py
```

### 4. 操作

1. キーとモードを選ぶ
2. 「開始」ボタンを押す(または F6)
3. 止めたいときは「停止」ボタンか F6

## 注意

- macOSの場合は「システム設定 → プライバシーとセキュリティ → アクセシビリティ」でターミナル/Pythonに許可が必要です
- 一部のゲーム(アンチチート搭載)ではソフトウェアによるキー入力が認識されない場合があります
